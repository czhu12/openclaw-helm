== Setup ==

1. Run onboarding:

  kubectl exec -it deploy/{{ include "openclaw.fullname" . }}-cli -n {{ .Release.Namespace }} -- node dist/index.js onboard --no-install-daemon

2. Add channels (examples):

  # WhatsApp (interactive QR login)
  kubectl exec -it deploy/{{ include "openclaw.fullname" . }}-cli -n {{ .Release.Namespace }} -- node dist/index.js channels login

  # Telegram
  kubectl exec -it deploy/{{ include "openclaw.fullname" . }}-cli -n {{ .Release.Namespace }} -- node dist/index.js channels add --channel telegram --token <BOT_TOKEN>

  # Discord
  kubectl exec -it deploy/{{ include "openclaw.fullname" . }}-cli -n {{ .Release.Namespace }} -- node dist/index.js channels add --channel discord --token <BOT_TOKEN>

3. Open a shell on the CLI pod anytime:

  kubectl exec -it deploy/{{ include "openclaw.fullname" . }}-cli -n {{ .Release.Namespace }} -- bash

{{- end }}
