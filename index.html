OpenClaw Helm Chart

Multi-channel AI gateway with extensible messaging integrations

chart: 0.1.0 app: 2026.2.13

Install

helm repo add openclaw https://czhu12.github.io/openclaw-helm
helm repo update
helm install openclaw openclaw/openclaw

Configure

helm install openclaw openclaw/openclaw \
  --set secret.data.OPENCLAW_GATEWAY_TOKEN=<your-token> \
  --set image.repository=your-registry/openclaw \
  --set image.tag=latest \
  --set ingress.enabled=true

All Values

helm show values openclaw/openclaw
